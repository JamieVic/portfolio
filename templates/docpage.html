HTML Documentation

Introduction

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

"HyperText" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

HTML uses "markup" to annotate text, images, and other content for display in a Web browser. Tags in HTML are case insensitive. That is, they can be written in uppercase, lowercase, or a mixture.

Additionally, HTML can be used alongside other programming languages to provide styling & functionality including:

  • CSS
  • JavaScript
  • Bootstrap
  • SASS
  • LESS
Headings

Headings are used to visually title an article or section of a document.

There are six levels of headings which differ in size and importance.

<h1>First important</h1> <h2>Second important</h2> <h3>Third important</h3> <h4>Fourth important</h4> <h5>Fifth important</h5> <h6>Sixth important</h6>
Paragraphs

Paragraphs contain the majority of text-based content on web pages.

Just like headings, paragraph tags will include white space margins so paragraphs aren't crammed together.

<p>This is a paragraph</p>
Links

HTML Links allow users to visit another page from one web page.

By default, links are distinguished by their blue text colour and underlined text decoration.

<a>This is a link</a>
Images

Images can be included in web pages very easily.

The image tag closes itself, meaning you don't have to include an ending tag.

<img src="link.png">
Lists

Lists are used to present text in a more organised fashion.

There are two types of lists that can be used; ordered and unordered.

<ul><li>This is an unordered list item</li></ul> <ol><li>This is an ordered list item</li></ol>
